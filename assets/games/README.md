# Images des jeux favoris

Certaines cartes de la section « Mes jeux préférés » utilisent une image locale
(champ `img` dans `data/favgames.js`). Dépose les fichiers ici avec ces noms exacts :

| Jeu                       | Fichier attendu           |
|---------------------------|---------------------------|
| Minecraft                 | `minecraft.jpg`           |
| Clair Obscur : Expédition 33 | `expedition33.jpg`     |
| Fortnite                  | `fortnite.jpg`            |

- Format bannière conseillé (paysage, ~16:9), en `.jpg`.
- Si une image est absente, la carte retombe proprement sur son emoji + dégradé
  (aucune carte cassée).
- Les autres jeux (TF2, Rust, Garry's Mod, DBZ Kakarot, YOMI Hustle, SCP: SL)
  chargent leur bannière Steam automatiquement, rien à déposer.
